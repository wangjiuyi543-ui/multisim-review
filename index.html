先明确考试要到什么程度

这部分只顺带考几个客观题，所以目标不是“会完整使用 Multisim”，而是建立最小理解框架。

基本操作：1—12

这一组别钻软件版本差异，考试通常按总结 PPT 的菜单名称出题。

1

绘制一个简单电阻电路

记流程
Place 放置电阻、电源和接地；Wire 将它们连接。

把 Multisim 想成“虚拟实验台”：先拿元件，再接线，最后运行。接地 Ground 是零电位参考，不是装饰。

2

添加新元件

记菜单
Place → Component Browser → 搜索元件 → 放到原理图。

Component Browser 就是元件库的搜索窗口。通常按“元件组 Group—元件族 Family—具体元件 Component”查找。

3

查看元件属性

记入口
选中元件，打开 Properties；也可右击元件。

属性里通常能看到数值、编号、模型、封装等。双击元件本质上也是打开属性。

4

导入其他电路图

记菜单
File → Import → 选择需要导入的电路文件。

这只是识别“导入”属于 File 菜单，不需要掌握文件兼容和版本转换。

5

查看元件连接关系

理解一下
按题库表述：使用 Wire 工具，或通过 Place 相关功能查看连接。

真正判断“是否电气连接”，主要看导线、连接节点圆点和网络名称。两根线只是在图上交叉，不一定连通。

客观题若完全照总结 PPT 出，就选择 Wire / Place；不要因为软件细节去和题库答案较劲。
6

设置元件数值

记入口
选中元件，在 Properties 中修改参数。

例如把电阻设为 1 kΩ、电容设为 1 μF。注意 M 是兆，m 是毫，大小写不同。

7

创建元件库文件

记工具名
使用 Component Editor 创建自定义元件，并保存到元件库。

Component Editor 相当于“元件制作器”，把符号、引脚、模型和封装等信息组合成一个可复用元件。

8

添加万用表

记仪表
Place → Measurement → Multimeter 或 Voltmeter。

电压表测两点间电压，要并联在被测两端；理想电压表内阻很大。

9

设置万用表参数

记档位
在 Properties 中选择电压档、电流档、电阻档以及 AC / DC。

AC 表示交流测量，DC 表示直流测量。题目一般只考“有哪些档位”，不考真实仪表误差。

10

添加电流表

容易错接
Place → Measurement → Current Meter 或 Ammeter。

电流表必须与被测支路串联，这样表中的电流就是支路电流。理想电流表内阻近似为 0。

电流表若并联在电源两端，近似造成短路；电压表若串联，近似造成断路。
11

导出 Probe 测量结果

记路径
Simulate → Plot Results → Save。

Probe 是探针，用来采集节点电压或支路电流；Plot Results 是把结果画出来和保存的窗口。

12

把 Probe 结果与其他参数比较

理解图像
在 Plot Results 中把多个变量绘制到同一张图上。

例如把输入电压和输出电压放在同一图里，就能比较幅值、相位和变化趋势。

2

基本概念：13—18

13

EDA 技术的三个阶段

纯背诵
20 世纪 70 年代 CAD；80 年代 CAE；90 年代以后 EDA。

可理解为：从“计算机帮你画图”，发展到“帮你分析”，再发展到“自动化完成更完整的电子设计流程”。

速记：70 CAD，80 CAE，90+ EDA。
14

设计工具箱的三种文件类型

纯背诵
工程文件、图页文件、支电路文件。

工程文件管理整个项目；图页文件是一张原理图；支电路文件是可重复调用的子模块。

15

ANSI 与 DIN

对比记忆
ANSI 是美国标准；DIN 是欧洲标准；我国元件符号与欧洲模式相同或接近。

最典型的视觉区别是电阻符号：美国常见锯齿形，欧洲常见矩形。

16

虚拟元器件与真实器件

高频辨析
虚拟器件参数可自由设置、无额定值限制、不能直接用于 Ultiboard PCB；真实器件对应具体型号，参数和额定值受限制，可关联实际封装。
虚拟器件
为仿真服务
参数可任意设置
通常无额定值限制
一般不能直接制 PCB
真实器件
对应厂家型号
参数由型号决定
有额定值限制
可带实际封装信息
题库中特别强调：电源虽然可能列在现实元器件栏中，但在仿真意义上属于虚拟元器件。
17

平坦式与层次式原理图

一定要分清
平坦式：同一逻辑层次，只是分布在多张图纸；层次式：存在顶层和子原理图的上下级关系。
口诀：平坦式“同层分页”，层次式“顶层分块、下层实现”。
18

离页连接器

条件题
用于不同图页之间的电气连接；图页须在同一电路文件夹中，连接器名称必须相同。

它相当于把跨页的一根长导线改成“同名网络标签”。Page1 的 DATA 与 Page2 的 DATA，被软件视为同一个网络。

3

仿真分析：19—20

19

Multisim 的两类仿真分析方法

理解用途
① 虚拟仪表直接测量；② 内置仿真分析法。
虚拟仪表法
像真实实验：放电压表、电流表、示波器，运行后直接读数。适合少量未知量。
内置分析法
让软件统一计算：直流工作点、交流扫描、瞬态、傅里叶、噪声、失真等。适合复杂电路。
类比：虚拟仪表法是“拿仪器测”；内置分析法是“让软件批量算”。
20

AC Sweep 是什么

核心理解
交流扫频分析：在一定频率范围内计算电路的频率响应；直流源置零，交流源作为正弦信号，输出与输入同频。

软件不是只测一个频率，而是让频率从低到高变化，逐点求输出，于是得到“响应随频率怎样变化”的曲线。

输入频率 f 改变 → 每个 f 都求一次稳态输出 → 得到幅频 / 相频特性
输出与输入同频，改变的是幅值和相位，不是产生另一个频率。
4

电感与电容的频率特性：21—22

这两题不要逐字硬背。记住两个公式，所有结论自然出来。

21

电感的交流频率特性

公式推出
频率越高，感抗越大，电感电压越大，电流越小；电感通直流、阻交流，通低频、阻高频。
XL = ωL = 2πfL

频率 f 增大，感抗 XL 就增大，所以电感对高频电流的阻碍更强。直流时 f=0，理想电感在稳态相当于短路。

最稳的记法：电感“频率越高，越难通过”。
22

电容的交流频率特性

公式推出
频率越高，容抗越小，电容电压越小，电流越大；电容通交流、隔直流，通高频、阻低频。
XC = 1/(ωC) = 1/(2πfC)

频率 f 增大，容抗 XC 反而减小，所以高频更容易通过。直流时 f=0，容抗趋向无穷，理想电容在稳态相当于开路。

最稳的记法：电容“频率越高，越容易通过”。

电感与电容一眼对照

元件阻抗大小频率升高直流稳态记忆
电感 LXL=2πfL感抗增大短路通低阻高
电容 CXC=1/(2πfC)容抗减小开路通高阻低
5

Bode Plotter：23—24

23

Bode Plotter 的功能

看懂用途
产生宽频率范围的扫描信号，用于测量电路的幅频特性和相频特性。

它关心的不是某一个时刻的波形，而是“电路对不同频率的信号分别有什么反应”。常用于滤波器、放大器等。

24

Bode Plotter 的两种模式

术语对应
Magnitude：幅度比随频率变化；Phase：相位差随频率变化，单位为度。
Magnitude
输出相对输入被放大或衰减了多少。常用比值或 dB 表示。
Phase
输出相对输入超前或滞后多少角度，单位 °。
Magnitude：|Uout/Uin|　　Phase：φout − φin
英语直接对应：Magnitude = 大小；Phase = 相位。

25

Basic 库包含什么

列举题
基本虚拟器件、定额虚拟器件、电阻、电位器、电容、电感、变压器等。
